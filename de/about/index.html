Über Toolhub

Was Toolhub ist

Toolhub ist eine Sammlung kostenloser Entwickler- und Alltags-Tools, die komplett in deinem Browser laufen. Keine Anmeldung, kein Konto, kein Tracking, keine serverseitige Verarbeitung. Daten einfügen, Ergebnis bekommen, Tab schließen — nichts wird gespeichert oder übertragen.

Warum

Die meisten Utility-Seiten im Netz sind entweder mit Werbung vollgepflasterte Parking-Pages oder schicken deine Daten durch vierzehn Drittanbieter-Tracker, bevor du überhaupt klickst. Toolhub ist die Alternative: eine Seite, ein Tool, läuft lokal, lässt dich in Ruhe.

Wer

JXXR1, unabhängiger Maintainer. Keine Firma, keine Finanzierungsrunde, keine Investoren. Toolhub fing als Nebenprojekt an — eine einzige Seite, die ein Tool gut macht — und ist von dort gewachsen. Erreichbar über GitHub.

Wie es funktioniert

Die Seite ist statisches HTML, gehostet auf GitHub Pages. Jedes Tool läuft als JavaScript in deinem Browser — keine API-Aufrufe, keine serverseitige Berechnung, keine Daten verlassen dein Gerät, außer es ist explizit angemerkt (zum Beispiel lädt das YouTube-Thumbnail-Tool ein Bild von YouTubes CDN; alles andere ist rein lokal).

Open Source

Der komplette Quellcode liegt auf https://github.com/JXXR1/Toolhub. Beiträge, Bug-Reports und Tool-Ideen sind über GitHub Issues willkommen.

Kein KI-Schrott

Hilfetexte und Übersetzungen sind nach Möglichkeit von Menschen geprüft, nicht aus einem LLM kopiert. Wenn dir eine Übersetzung robotisch oder falsch vorkommt, öffne ein Issue — das ist eine schnelle Korrektur und macht die Seite für alle besser.
