Notenrechner

Gewichtete Klassenstufen-Note berechnen. Kategorien (Hausaufgaben / Klausuren / Projekte / Abschlussprüfung) mit Gewichtungen festlegen, Punkte eintragen, laufende Note sehen. Was-wäre-wenn-Modus: 'Was brauche ich in der Abschlussprüfung für eine 1?'

Wozu ist das gut?

Mitte des Semesters macht jede:r Schüler:in dieselbe Panikrechnung: „Ich hab in der Klausur eine 2,3 und in den Hausaufgaben durchgehend 1en — was brauche ich in der Abschlussprüfung, um die 2 zu halten?" Genau das löst dieser Rechner. Stelle die Kategorien wie in deiner Leistungsbewertung ein (Hausaufgaben, Tests, Projekte, Abschlussprüfung), gib das Gewicht jeder Kategorie ein, trage deine Punkte ein — und du siehst deine laufende Note mit Buchstaben. Im Was-wäre-wenn-Modus rechnet das Tool rückwärts: Bei welchem Wert in der ausstehenden Prüfung erreichst du das Ziel?

So funktioniert's

  • Kategorien entsprechend deiner Bewertungsrichtlinie anlegen. Gewichte sollten 100 ergeben — oben steht eine Warnung, wenn nicht.
  • Punkte als „erreicht / möglich" eintragen (z. B. 18 / 20). Innerhalb einer Kategorie werden Punkte summiert, nicht Prozentwerte gemittelt. So zählt eine 50-Punkte-Klausur mehr als ein 10-Punkte-Quiz — was Lehrkräfte üblicherweise so machen.
  • Was-wäre-wenn-Modus. Wähle die noch ausstehende Kategorie, gib deine Zielnote in % ein, sieh die nötige Leistung. Über 100 %: nicht möglich. Unter 0 %: schon erreicht.
  • Notenskala. Standard ist US-Skala mit Plus/Minus. Du kannst auf einfache A/B/C/D/F, UK-Skala oder eigene Schwellen umschalten — auch direkt im Editor.
  • Stand in URL speichern schreibt deine Daten ins Hash-Fragment. Bookmark setzen, Link an dich schicken — die Daten verlassen den Browser nicht (Hash-Fragmente werden nicht an Server geschickt).

Typische Fehler

  • Gewichteter ≠ einfacher Mittelwert. 20 % HÜ-Gewicht und 30 % Klausur: 100 in HÜ + 80 in Klausur = (100×20 + 80×30) / 50 = 88, nicht 90.
  • Punkte- vs. Prozent-Mittel. Dieses Tool summiert Punkte je Kategorie. Wenn jede Klausur gleich zählen soll (unabhängig von der Maximalpunktzahl), trage Prozente als „X / 100" ein.
  • Nicht-bewertete Kategorien. Die laufende Note nimmt nur die Kategorien mit Eingaben. Der „verwendetes Gewicht"-Hinweis zeigt, wie viel der 100 % im Spiel sind.
  • Streichnote / niedrigste weg. Einfach nicht eintragen.
  • Bonuspunkte. Trage > maximal mögliche ein (z. B. 12 / 10). Funktioniert.
  • Datenschutz. Nichts wird gesendet. URL-Save schreibt nur ins Hash, das Browser nicht an Server übertragen.
