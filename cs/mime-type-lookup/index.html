MIME Type Lookup

Hledej MIME typy podle přípony nebo typu. ~120 běžných typů — image, video, audio, application, text, font.

K čemu to slouží?

MIME (Multipurpose Internet Mail Extensions) types popisují typ obsahu — image/png, application/json, text/html. HTTP servery je posílají v Content-Type headeru. Tenhle nástroj má databázi ~120 běžných typů; hledej podle přípony nebo MIME typu.

Kdy to použít

  • Nastavení správného Content-Type v server configu (Nginx, Apache).
  • Validace, jestli file upload posílá správný MIME.
  • Debug, proč prohlížeč stahuje soubor místo renderování.
  • Příprava Accept hlavičky pro API request.

Časté chyby

  • Přípona ≠ obsah. Útočník může přejmenovat .exe na .jpg. Server by měl MIME odvodit z obsahu (magic bytes), ne z přípony.
  • application/octet-stream. Generický „nevím, co to je" fallback. Prohlížeč stáhne jako soubor.
  • charset parameter. text/html; charset=UTF-8 — bez toho prohlížeč hádá encoding.
  • JSON je application/json, ne text/json (i když některé servery vrací text/json).
  • Vendor types. application/vnd.api+json (JSON:API), application/ld+json (JSON-LD) — jsou specializované.
