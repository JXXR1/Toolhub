JSON formátovač

Naformátuj, over a minifikuj JSON okamžite. Chyby zvýraznené s riadkom a stĺpcom.

Zadaj vstup vyššie, aby si videl výsledok.

Načo to slúži?

JSON formátovač naformátuje JSON s konzistentným indentom (alebo ho minifikuje na jeden riadok), zároveň validuje syntax. Pri chybe ti ukáže presný riadok a stĺpec, aby si nehľadal čiarku navyše hodinu.

Kedy to použiť

  • API odpoveď, ktorú dostaneš ako jeden dlhý riadok, premeniť na čitateľnú.
  • Validácia, či JSON, ktorý práve píšeš, parse-uje.
  • Minify pred poslaním cez network (alebo do localStorage).
  • Príprava pretty-printed JSON-u do dokumentácie.

Časté chyby

  • Trailing comma. JSON nepovoľuje čiarku za posledným prvkom (na rozdiel od JS objektov).
  • Komentáre. Štandardný JSON ich nemá. Použij JSON5 alebo JSONC, ak ich nutne potrebuješ.
  • Single quotes. JSON keys aj string values musia byť v double quotes.
  • NaN / Infinity / undefined. Nie sú validné JSON — JS to dovoľuje, ale JSON.parse zlyhá.
  • Veľké čísla. JSON nedefinuje presnosť — 64-bit JS čísla strácajú presnosť cez 2^53.
