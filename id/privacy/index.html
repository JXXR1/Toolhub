Kebijakan Privasi

Terakhir diperbarui: 11 May 2026

Versi singkat

Alat berjalan sepenuhnya di browser-mu — apa pun yang kamu ketik di dalamnya tidak dikirim ke mana pun. Kami mengukur kunjungan secara agregat dan anonim dengan Plausible Analytics (tanpa cookie). Jika kamu memilih, kami juga bisa menampilkan iklan display melalui Google AdSense yang menggunakan cookie. Kamu bisa menolak iklan di banner persetujuan; sisa situs bekerja sama saja.

Apa yang kami kumpulkan

  • Input alat: tidak ada. Apa pun yang kamu tempel ke sebuah alat tetap di browser-mu. Tidak ada alat yang mengirim data ke server.
  • Statistik kunjungan: Plausible menghitung tampilan halaman, referrer, negara, dan tipe perangkat — anonim, agregat, tanpa cookie. Kebijakan data Plausible menjelaskan apa yang mereka kumpulkan dan tidak.
  • Jika kamu menerima iklan: Google AdSense menempatkan cookie dan bisa menggunakannya untuk mempersonalisasi iklan. Praktik Google tunduk pada kebijakan privasi iklan Google.
  • Jika kamu menolak iklan: skrip iklan tidak dimuat, cookie iklan tidak ditempatkan, dan blok iklan menghilang dari halaman.

Cookie

Situs ini menggunakan satu entri localStorage (toolhub:consent) untuk mengingat preferensi iklanmu, plus entri theme untuk preferensi gelap/terang. Tidak ada yang meninggalkan browser-mu.

Plausible tidak menggunakan cookie. Google AdSense menggunakan cookie hanya jika kamu menerima — kamu selalu bisa mengubah pilihanmu via tautan persetujuan di footer.

Layanan eksternal

  • Plausible Analytics — ramah privasi, sesuai GDPR, di-host di EU. Tanpa data pribadi.
  • Google AdSense — hanya dengan persetujuan eksplisit. Memuat pagead2.googlesyndication.com dan bisa menempatkan cookie iklan.
  • GitHub Pages — host situs. GitHub secara default menyimpan log server (IP, timestamp, URL) sesuai pernyataan privasinya.
  • Alat YouTube Thumbnail — khusus jika kamu menggunakan alat ini, browser-mu mengambil gambar langsung dari i.ytimg.com (CDN YouTube). Tanpa otentikasi, tanpa upload. Detail di penanganan data.

Tautan afiliasi

Beberapa tautan di footer adalah tautan afiliasi — jika kamu mengklik dan mendaftar ke layanan yang direkomendasikan (misalnya penyedia hosting), Toolhub bisa mendapat komisi. Harga yang kamu bayar tidak berubah. Tautan afiliasi ditandai dengan rel="sponsored". Daftar lengkap di pengungkapan afiliasi.

Hak kamu

Berdasarkan GDPR / UK GDPR / CCPA / undang-undang privasi Slovakia, kamu bisa meminta akses atau penghapusan data pribadi yang kami simpan tentangmu. Kami tidak menyimpan apa-apa — tanpa backend, tanpa akun, tanpa database pengguna. Plausible hanya menyimpan data kunjungan agregat. Data iklan Google melalui saluran permintaan mereka sendiri.
