Konverter format gambar

Konversi gambar antara PNG, JPG, dan WebP di browser-mu. Drop, pilih format target, unduh. Tanpa upload, tanpa server.

Untuk apa ini?

Mengkonversi format container gambar — PNG ↔ JPG ↔ WebP — sepenuhnya di browser-mu lewat Canvas API. Gambar di-decode jadi pixel, lalu di-encode ulang ke format target. File tidak pernah meninggalkan perangkat; tanpa upload, tanpa pemrosesan di server.

Kapan digunakan

  • Sebuah situs atau app menolak WebP dan menuntut JPG/PNG.
  • Kamu publish di web dan ingin WebP untuk payload lebih kecil pada kualitas sama.
  • Kamu butuh PNG untuk mempertahankan tepi transparan (mis. logo di atas latar berwarna).
  • Kamu mengumpulkan screenshot — JPG biasanya lebih kecil dari PNG untuk konten fotografi.

Kesalahan umum

  • JPG tidak punya channel alpha. Saat mengonversi PNG transparan ke JPG, pixel transparan jadi putih (tool ini mengisi latar putih sebelum menggambar).
  • Re-encoding teks/screenshot ke JPG terlihat kabur. Tepi tajam dan blok warna datar sulit dikompres JPEG. Pertahankan PNG (atau WebP lossless) untuk screenshot UI.
  • Dukungan encoder WebP bervariasi. Chromium dan Firefox sudah lama; Safari sejak 14. Jika browser-mu tidak bisa encode WebP, konversi gagal diam-diam — coba target lain.
  • Slider kualitas tidak berlaku untuk PNG. PNG lossless dan mengabaikan argumen kualitas.
  • Metadata dibuang. Re-encoding via Canvas menghapus EXIF, profil ICC, dan copyright. Plus untuk privasi, tapi waspada jika workflow-mu butuh profil warna.
