EXIF Stripper

EXIF क्या है और इसकी परवाह क्यों करें?

हर photo जो आप एक phone या camera से लेते हैं, उसमें एक छिपा data block होता है — EXIF metadata — जो pixels के साथ file में embed होता है। ज़्यादातर लोग कभी इसे नहीं देखते। यह वहीं है: photo कहाँ ली गई थी उसके GPS coordinates (अक्सर कुछ meters तक accurate), जब shutter fired हुआ था आपके phone के clock पर exact second, आपके device का make और model, lens, कभी-कभी camera के owner field से आपका नाम। उस JPG को एक forum पर upload करें, WhatsApp पर भेजें, एक personal blog पर post करें, और वह data इसके साथ जाता है, जब तक कि platform इसे strip न करे।

यह tool आपके browser में directly EXIF पढ़ता है (कोई upload नहीं — page पर JavaScript bytes को parse करता है), आपको दिखाता है कि क्या है, और एक one-click download देता है उसी photo का जिसमें सभी metadata हटा दिए गए हैं। Original file कभी आपके device से बाहर नहीं जाती।

वास्तव में क्या expose होता है

  • GPS coordinates। यदि photo लेते समय location services on थे, तो latitude/longitude baked in हैं। Six decimal places लगभग 11 cm precision है — आपके home address की पहचान के लिए पर्याप्त।
  • Date और time। Second तक, camera के local timezone में।
  • Camera model + serial। कुछ cameras एक unique serial embed करते हैं जो multiple photos को एक ही physical body से जोड़ता है।
  • Lens, aperture, ISO, focal length। Photographers के लिए उपयोगी; leak करना harmless।
  • Editing software। "Photoshop CC 2023" बताता है कि image processed है, कभी-कभी editor की identity के साथ।
  • Owner / artist / copyright fields। कुछ cameras आपको एक default नाम set करने देते हैं। यह हर photo के साथ यात्रा करता है।

आपको कब EXIF strip करना चाहिए

  • अपने घर, कार्यस्थल, या किसी ऐसी जगह की photo post करने से पहले जो आप map पर pin नहीं करना चाहते।
  • एक "मैंने यह कहाँ लिया" picture publicly share करने से पहले जब आप कहाँ रहते हैं यह reveal नहीं करना चाहते।
  • Journalists, lawyers, support groups, या किसी भी sensitive context में किसी को photos भेजते समय।
  • Whistleblower / source-protection workflows — किसी भी image evidence को upload करने से पहले metadata strip करें।
  • एक marketplace पर listing करते समय (OLX, Facebook Marketplace) — आपके sofa की photo आपका address reveal कर सकती है।

यह tool क्या करता है और क्या नहीं

  • करता है: JPG files से EXIF पढ़ता है। आपको GPS, camera, timestamp, lens, software दिखाता है। JPG को बिना किसी APP1 (EXIF), APP13 (Photoshop IRB / IPTC), या APP14 (Adobe) segments के re-emit करता है।
  • करता है: पूरी तरह आपके browser में चलता है। कुछ भी upload नहीं होता। File का कोई server log कभी नहीं होता।
  • नहीं करता: RAW formats (CR2, NEF, ARW) में EXIF handle — उनके लिए desktop tool चाहिए। PNG और WebP shaayad ही EXIF carry करते हैं।
  • नहीं करता: Pixels में baked metadata strip करना (जैसे image में burned visible timestamp, या एक watermark)।
  • नहीं करता: JPG को re-encode। Pixel data byte-identical है; केवल metadata segments dropped होते हैं, इसलिए quality unchanged है।

आम गलतियाँ

  • Social networks अलग-अलग हैं। Facebook और Twitter upload पर EXIF strip करते हैं; Instagram historically भी करता था। पर "image hosting" sites, forums, और personal blogs अक्सर नहीं करते। हर platform को untrusted मानें जब तक आपने test न किया हो।
  • Screenshots में आमतौर पर कोई EXIF नहीं होता। Tool आपको बताएगा यदि यह पहले से ही clean है।
  • Cloud sync original रखता है। आपके download किए JPG से EXIF strip करना iCloud / Google Photos में मौजूद copy को नहीं बदलता।
  • Thumbnail survive कर सकता है। कुछ software EXIF block के अंदर एक छोटा JPEG preview store करते हैं; यह tool पूरा EXIF segment drop करता है, इसलिए वह preview भी जाता है।
