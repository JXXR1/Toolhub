गोपनीयता नीति

अंतिम बार अद्यतन: 11 May 2026

संक्षिप्त संस्करण

Tools पूरी तरह आपके browser में चलते हैं — आप उनमें जो भी टाइप करते हैं वह कहीं नहीं भेजा जाता। हम Plausible Analytics के साथ समग्र, अनाम विज़िट गिनते हैं (cookies नहीं)। अगर आप opt-in करना चुनते हैं, तो हम Google AdSense के माध्यम से display विज्ञापन भी दिखा सकते हैं, जो cookies का उपयोग करता है। आप consent banner पर विज्ञापन अस्वीकार कर सकते हैं; बाकी साइट दोनों ही स्थितियों में समान रूप से काम करती है।

हम क्या एकत्र करते हैं

  • Tool inputs: कुछ नहीं। आप जो भी किसी tool में paste करते हैं वह आपके browser में ही रहता है। कोई भी tool server को डेटा नहीं भेजता।
  • विज़िट analytics: Plausible page views, referrer, देश, और device प्रकार गिनता है — अनाम रूप से, समग्र रूप में, cookies के बिना। Plausible की data policy बताती है कि वे क्या एकत्र करते हैं और क्या नहीं।
  • अगर आप विज्ञापन स्वीकार करते हैं: Google AdSense cookies set करता है और उनका उपयोग विज्ञापनों को personalize करने के लिए कर सकता है। Google की प्रथाएँ Google की विज्ञापन गोपनीयता नीति के तहत आती हैं।
  • अगर आप विज्ञापन अस्वीकार करते हैं: कोई विज्ञापन script load नहीं होती, कोई विज्ञापन cookie set नहीं होती, page से विज्ञापन slots हटा दिए जाते हैं।

Cookies

साइट स्वयं एक localStorage entry (toolhub:consent) का उपयोग आपकी विज्ञापन-सहमति के निर्णय को याद रखने के लिए करती है, साथ ही आपकी dark/light वरीयता के लिए एक theme entry। दोनों आपके browser से बाहर नहीं जाते।

Plausible कोई cookies उपयोग नहीं करता। Google AdSense cookies केवल तभी उपयोग करता है जब आप उन्हें स्वीकार करते हैं — आप footer में consent link के माध्यम से किसी भी समय अपना निर्णय बदल सकते हैं।

तृतीय-पक्ष सेवाएँ

  • Plausible Analytics — privacy-friendly, GDPR-compliant, EU-hosted। कोई व्यक्तिगत डेटा एकत्र नहीं।
  • Google AdSense — केवल स्पष्ट सहमति के साथ। pagead2.googlesyndication.com load करता है और विज्ञापन cookies set कर सकता है।
  • GitHub Pages — साइट host। मानक server logs (IP, timestamp, URL) GitHub द्वारा अपनी गोपनीयता वक्तव्य के अनुसार रखे जाते हैं।
  • YouTube Thumbnail tool — जब आप विशेष रूप से इस एक tool का उपयोग करते हैं, तो आपका browser सीधे i.ytimg.com (YouTube के CDN) से एक image fetch करता है। कोई प्रमाणीकरण नहीं, कोई upload नहीं। विवरण के लिए data handling देखें।

Cloudflare Web Analytics

हम Toolhub पर विज़िट गिनने के लिए Cloudflare Web Analytics का उपयोग करते हैं। यह एक गोपनीयता-संरक्षित analytics सेवा है जो cookies उपयोग नहीं करती, आपके browser का fingerprint नहीं लेती, और कोई व्यक्तिगत डेटा एकत्र नहीं करती। Cloudflare केवल page views, मोटे तौर पर देश, और referrer स्रोत को aggregate करता है। किसी व्यक्तिगत visitor को sessions के बीच ट्रैक नहीं किया जा सकता। डेटा का उपयोग केवल यह समझने के लिए होता है कि कौन से tools उपयोगी हैं और कौन सी भाषाएँ अपने audience तक पहुँच रही हैं।

अगर आप चाहें तो आप network level पर beacon को block कर सकते हैं — किसी भी tool की कार्यक्षमता पर असर नहीं पड़ता। Script static.cloudflareinsights.com से load होता है और एक config flag के पीछे gated है; अगर साइट operator ने अभी तक analytics सेट अप नहीं किया है, तो कोई request भी नहीं भेजी जाती।

Affiliate links

Footer के कुछ links affiliate links हैं — अगर आप उनके माध्यम से click करते हैं और हमारे द्वारा सुझाई गई सेवा (जैसे hosting providers) के लिए sign up करते हैं, तो Toolhub को एक referral मिल सकता है। आप जो क़ीमत चुकाते हैं वह नहीं बदलती। Affiliate links rel="sponsored" से tag किए जाते हैं। पूरी सूची के लिए affiliate disclosure देखें।

आपके अधिकार

GDPR / UK GDPR / CCPA / Slovak DPA के तहत, आप अपने पास रखे गए किसी भी व्यक्तिगत डेटा तक पहुँच या उसे हटाने का अनुरोध कर सकते हैं। हमारे पास कोई नहीं है — कोई backend नहीं, कोई अकाउंट नहीं, कोई user database नहीं। Plausible केवल समग्र विज़िट डेटा रखता है। Google का विज्ञापन डेटा उनके अपने अनुरोध माध्यमों द्वारा नियंत्रित होता है।
