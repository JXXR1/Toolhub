Image Resizer

Images को अपने browser में किसी भी width या height पर resize करें। Aspect ratio lock करें, social-media-friendly sizes पर scale down करें, तुरंत download करें। कोई upload नहीं।

🔒 Files never leave your browser — no upload, no server, no tracking.
Quality applies to JPG and WebP outputs only.
Pick an image to begin.

यह किसके लिए है?

यह tool एक PNG, JPG, या WebP image को आपके बताए pixel dimensions पर resize करता है। Image browser द्वारा decode होती है, target size पर high-quality smoothing के साथ एक canvas पर draw होती है, और उसी format में फिर से encode होती है। Output एक blob के रूप में download होता है। कुछ भी upload नहीं होता — resize पूरी तरह client-side होता है।

कब इस्तेमाल करें

  • Social-media-ready image बनाना (Twitter header 1500×500, OG card 1200×630, Instagram square 1080×1080)।
  • Blog या CMS पर upload करने से पहले एक बड़े phone photo को web-appropriate width पर trim करना।
  • Thumbnails बनाना — half-size या quarter-size presets numbers type किए बिना quick downscale देते हैं।
  • हर dimension में कुछ सौ px कम करके एक image को upload-size cap के नीचे squeeze करना।

आम गलतियाँ

  • Lock aspect ratio default पर ON है। अगर आपको arbitrary aspect पर crop करना है (जैसे portrait phone photo से strict 1200×630 OG card), तो lock uncheck करें — image fit होने के लिए stretch हो जाएगा। बिना distortion वाले real cropping के लिए, पहले किसी और tool में pre-crop करें।
  • Quality slider केवल JPG/WebP के लिए है। PNG losslessly re-encode होता है, इसलिए slider PNG output पर असर नहीं डालता।
  • बड़े canvases browser limits से टकराते हैं। ज़्यादातर browsers एक single canvas को ~16384×16384 px पर cap करते हैं (और mobile पर बहुत कम)। अगर आपकी image उससे बड़ी है, resize चुपचाप fail हो सकता है।
  • Upscaling detail नहीं जोड़ता। 500×500 से 2000×2000 तक resize करना pixels को stretch करता है — result blurry दिखता है। हमेशा अपने पास उपलब्ध highest-resolution source से शुरू करें।
