यह क्या है — और क्या नहीं

एक त्वरित reference currency converter। Source, target, amount डालें। Rates page में hard-coded हैं — कोई network call नहीं, कोई API key नहीं, कोई tracking नहीं। लगभग quarterly update होती हैं, दिखाई गई date के लिए valid हैं।

यह forex terminal नहीं है। असली bank rates हर second बदलती हैं और इनमें margin (retail में 0.5–3%), card network fees, और weekend markup शामिल होते हैं। Invoice भेजने या payment करने से पहले अपने bank, Wise, या public FX feed से current rate देखें।

हम live rates क्यों नहीं लाते

लगभग हर "free" live-rate API आपकी IP, referrer और query किया गया pair log करती है। Freelancer के लिए जो दिनभर tool खुला रखता है, हर refresh आपके client portfolio के बारे में एक और data point है। Static table यह leak पूरी तरह हटा देती है।

आम गलतियाँ

  • Pegs आज fixed लगते हैं, हमेशा नहीं रहते (CHF cap 2015 में अचानक गिरा)।
  • EUR के via routing से illiquid pairs पर rounding error बढ़ता है।
  • Mid-rate वह नहीं जो आपको मिलेगा: 1–4% spread/commission।
  • ARS और TRY एक महीने में 5–10% हिल सकते हैं।
  • Airport money changer: bank transfer से 5–10% खराब।

कब उपयोगी है

  • विदेशी currency में freelance quote draft करना — approximate।
  • आने वाला विदेशी invoice check करना।
  • दो websites पर अलग currencies में prices compare करना।
  • Wire transfer सही range में पहुँचा है verify करना।

इनके लिए नहीं: hedging, FX trading, tax filing (अपने tax authority का official rate use करें), या जहाँ 1–3% का फर्क मायने रखता हो।
