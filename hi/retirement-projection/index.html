Retirement Projection

अपनी retirement savings project करें: current balance, annual contributions, expected return, inflation, target year। Real (inflation-adjusted) और nominal balances देखें। Multiple return scenarios से stress-test करें।

Projections, not guarantees. Real returns vary, sequence-of-returns risk is real, inflation can spike. Use this for ballpark planning, not as investment advice.

यह किसके लिए है?

Retirement सबसे बड़े financial decisions में से एक है — और यह years पहले ही a few inputs से तय होती है: कितना save करते हैं, क्या earn होता है, और कितने time grow होने को मिलता है। यह tool उन inputs को target retirement age तक project करता है, फिर math को reverse में चलाकर दिखाता है कि चुने gए withdrawal level पर वह pot कितने साल चलेगा। Nominal balance और real balance (inflation-adjusted, आज के money में) दोनों दिखाता है।

Math

Accumulation phase: balance × (1 + return) + annual contribution। Inflation-adjusted value: nominal ÷ (1 + inflation)^years।

आम गलतियाँ

  • Sequence-of-returns risk real है. समान average return वाले दो retirees बहुत अलग जगह पहुँच सकते हैं अगर एक bad decade से ठीक पहले retire हो जाए। यह tool model नहीं करता।
  • Inflation 2.5% नहीं है. India: 1990s में high, हाल में 5–7% range। कई values से test करें।
  • Nominal return vs real return. Inflation और fees के बाद 4–5% real एक honest expected number है।
  • Withdrawals taxable हैं. NPS, EPF withdrawals पर अलग tax rules लागू होते हैं — net amount देखें।
