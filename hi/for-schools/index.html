स्कूलों के लिए Toolhub

कक्षा में Toolhub

Toolhub छोटे in-browser tools का एक सेट है जिसका छात्र बिना अकाउंट बनाए, बिना ट्रैक हुए, और बिना विज्ञापन या affiliate funnels पर redirect हुए उपयोग कर सकते हैं। हर tool पूरी तरह browser में चलता है, जिसका मतलब है छात्र का input कभी स्कूल network से बाहर नहीं जाता — कोई backend है ही नहीं जिसे call किया जा सके।

अगर आप प्राथमिक, माध्यमिक या उच्च शिक्षा में पढ़ाते हैं और एक त्वरित utility चाहिए (regex tester, color converter, Base64 encoder, security पाठ के लिए password generator), तो Toolhub इस तरह बना है कि इसे projector पर दिखाना और कक्षा के साथ साझा करना सुरक्षित है।

पाठ्यक्रम से जुड़ाव

  • Computer science: pattern matching के लिए regex tester, डेटा के पाठों के लिए Base64 encoder, integrity की बात के लिए hash generator, networking इकाइयों के लिए CIDR calculator।
  • Design और digital media: color models पर चर्चा के लिए color converter और color picker, accessibility के लिए WCAG contrast checker।
  • Security awareness: entropy और password मज़बूती की चर्चा के लिए password generator, token के अंदर वास्तव में क्या है यह दिखाने के लिए JWT decoder।
  • गणित और स्वास्थ्य: percentage calculator, unit converter, और BMI calculator (BMI page पर "यह चिकित्सीय सलाह नहीं है" का अपना notice है)।

Self-hosted विकल्प

अगर आपका स्कूल network बाहरी साइटों को block करता है या आप पूरा नियंत्रण चाहते हैं, तो पूरी साइट लगभग 5 MB की है और Progressive Web App के रूप में offline काम करती है। आप इसे स्कूल intranet पर mirror कर सकते हैं — यह HTML, CSS और JavaScript का एक static folder है, serve करने के लिए कोई build step ज़रूरी नहीं: बस फ़ाइलों को किसी भी web server के पीछे रख दीजिए।

पूरा source code https://github.com/JXXR1/Toolhub पर है।

Filter-friendly

Toolhub को इस तरह design किया गया है कि वह स्कूल के web filters के साथ अच्छी तरह काम करे:

  • कोई embedded social media widget नहीं।
  • कोई chat widget या live-chat overlay नहीं।
  • अन्य साइटों पर कोई auto-redirect नहीं।
  • कोई video या audio जो अपने आप चले।
  • SafeSearch-friendly सामग्री — कोई adult tools नहीं, कोई जुआ नहीं, कोई crypto affiliate placement नहीं।

IT admins के लिए trust signals

अगर आपके स्कूल का network filter किसी नए domain को अनुमति देने से पहले verification मांगता है, तो ये सार्वजनिक, third-party signals हैं जिन्हें आप check कर सकते हैं:

  • Google Safe Browsing: Toolhub का status verify करें — Google का स्वचालित phishing/malware classifier। Toolhub clean list में है।
  • VirusTotal scan: नवीनतम scan परिणाम देखें — स्वतंत्र third-party multi-engine scan (70+ antivirus engines)।

Toolhub GitHub पर open source भी है — IT admins domain को approve करने से पहले कोड की हर पंक्ति की जांच कर सकते हैं।

शिक्षकों के लिए संपर्क

अगर आप कक्षा में Toolhub का उपयोग कर रहे हैं और मुझे बताना चाहते हैं, अपने विषय के लिए कोई tool सुझाना चाहते हैं, या किसी ऐसी भाषा के लिए अनुवाद में योगदान देना चाहते हैं जिसे हम अभी तक अच्छी तरह cover नहीं कर पाए — कृपया संपर्क page के माध्यम से संपर्क करें। मूल वक्ताओं से बड़े पैमाने पर अनुवाद योगदान — विशेष रूप से कम सेवा वाली भाषाओं के लिए — बहुत स्वागत है।
