Trình chỉnh sửa Bảng Markdown

Chỉnh sửa bảng trực quan — hàng, cột, căn lề theo cột — và sao chép Markdown tương thích GitHub.
Click any cell to edit. Use the dropdown row to set per-column alignment.
Nhập dữ liệu ở trên để xem kết quả.

Công cụ này để làm gì?

Bảng Markdown nhanh đọc nhưng đau viết — căn lề cell với pipe và dash bằng tay khó. Tool này cho phép bạn chỉnh sửa bảng một cách trực quan (thêm/bớt hàng và cột, đặt căn lề cột) và sao chép Markdown sẵn-dùng tương thích GitHub.

Khi nào nên dùng

  • Tạo bảng so sánh trong README.
  • Chèn bảng dữ liệu vào pull request hoặc issue.
  • Chỉnh sửa bảng phức tạp mà không phải đếm pipe.

Lưu ý thường gặp

  • GFM khác Markdown cổ điển. Bảng không phải là một phần của Markdown gốc của John Gruber — chúng là extension của GFM. Một số renderer (kindle, một số e-reader) không xử lý chúng.
  • Pipe trong cell cần escape. \| sẽ render là pipe literal. Tool này xử lý điều đó tự động.
  • Bảng không hỗ trợ rowspan/colspan. Đối với layout phức tạp, dùng HTML trực tiếp.
