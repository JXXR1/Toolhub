XML Formatter

Format và minify XML. Xác thực well-formedness với dòng và cột khi có lỗi.

Nhập dữ liệu ở trên để xem kết quả.

Công cụ này để làm gì?

XML như JSON nhưng dài hơn — tag với attribute, content text, nested element. Tool này pretty-print XML với indent thích hợp hoặc minify nó để loại bỏ whitespace không cần thiết. Lỗi well-formedness được flag với số dòng và cột.

Khi nào nên dùng

  • Đọc response SOAP API.
  • Format config file XML cũ.
  • Xác minh well-formedness của XML hand-written.

Lưu ý thường gặp

  • Well-formed không phải valid. Well-formed có nghĩa cú pháp XML là OK; valid có nghĩa nó khớp với schema (DTD, XSD). Tool này check well-formed, không validate.
  • Whitespace có thể có ý nghĩa. Trong text content và một số attribute, whitespace không thể remove an toàn. Tool này thận trọng.
  • Namespace. Prefix tag (ns:tag) cần khai báo namespace match. Tool này không validate namespace — chỉ format.
