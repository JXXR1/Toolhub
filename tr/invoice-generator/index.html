Fatura Oluşturucu

Markdown + yazdırmaya hazır HTML olarak temiz fatura oluştur. Formu doldur, canlı önizleme gör, tarayıcının yazdırma penceresinden PDF olarak kaydet. Yükleme yok, kayıt yok.

Niye var


Freelancer veya tek kişilik şirket olarak ara sıra düzgün bir fatura kesmen gerek — ama SaaS aboneliği gereksiz. Bu araç markdown + yazdırılabilir HTML fatura üretir, tarayıcıda hesaplar. Yazdırma diyaloğundan PDF olarak kaydet.


Kullanım


    

  1. Gönderen / Alıcı doldur.
    2. 

  2. Numara, tarihler, para birimi.
    3. 

  3. Kalemleri ekle.
    4. 

  4. KDV ülkesi seç (standart oranı yükler) veya özel oran.
    5. 

  5. "URL'ye Kaydet" — şirket bilgilerin URL parçasına gider.
    6. 

  6. "Yazdır / PDF" → diyalogdan "PDF olarak kaydet".
    7. 



KDV — önce oku


    

  • Yerleşik olduğun ülke kuralları belirler.
    • 

  • Müşterinin yeri (yurtiçi, AB B2B, AB B2C, AB dışı) her şeyi değiştirir.
    • 

  • Hizmet tipi oranı belirler.
    • 

  • AB B2B'de reverse-charge — 0 % fatura + zorunlu ibare.
    • 

  • AB B2C dijital hizmette yılda 10.000 € üzeri için OSS.
    • 



URL parçası neyi saklar


Gönderen/Alıcı blokları, para birimi, KDV, notlar. Parçalar sunucuya gitmez ama URL'yi alan herkes verileri görür. Kendine yer imi ekle; paylaşma.


Sık yapılan hatalar


    

  • Boşluksuz ardışık numaralandırma (genelde yasal zorunluluk).
    • 

  • PDF ve kaynağı sakla.
    • 

  • Görüntülenen para birimi ≠ tahsil para birimi — açıkça yaz.
    • 

  • Gecikme faizi maddesi faturanın üstünde.
    • 

  • AB B2B'de iki KDV numarası da zorunlu.
    •
