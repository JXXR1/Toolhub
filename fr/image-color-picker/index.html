À quoi ça sert ?

Cet outil transforme n'importe quelle image uploadée en une source de couleur cliquable. Survolez n'importe où pour lire le pixel exact sous le curseur en hex, RGB et HSL — avec une loupe qui zoome 8× pour picker des pixels avec précision sans plisser les yeux. Cliquez pour verrouiller une couleur dans une palette personnalisée : construisez la palette en cliquant plusieurs points. Cliquez sur une nuance dans la palette pour la retirer. L'image ne quitte jamais votre navigateur.

Quand l'utiliser

  • Extraire une teinte précise d'une photo pour l'utiliser dans un design.
  • Faire correspondre une couleur d'accent UI à un actif de marque dont vous n'avez pas le hex.
  • Extraire les couleurs individuelles d'une marque depuis une image de logo, clic après clic.
  • Construire une palette moodboard à partir d'un photogramme ou d'une œuvre.
  • Vérifier que deux régions d'une image sont vraiment la même couleur (vs sembler similaires mais différer dans les pixels).

Comment fonctionne la précision

L'échantillonnage 1-pixel est exact mais bruyant — bords anti-aliasés, artefacts JPEG et rendu sous-pixel font diverger légèrement les voisins. La loupe 8× vous laisse voir et cliquer le pixel spécifique visé, plutôt que de vous fier à la moyenne. Pour l'extraction de couleurs de marque cela compte ; pour un moodboard vous pouvez l'ignorer.

Pièges courants

  • Anti-aliasing sur le texte. Les lettres ont des bords doux qui ne représentent pas la couleur "encre" sous-jacente. Cliquez au milieu d'un trait épais, pas sur un bord.
  • Artefacts JPEG. Les photos très compressées dérivent en couleur dans des régions apparemment plates. Si vous avez besoin d'une couleur propre, préférez les sources PNG.
  • Image redimensionnée pour l'affichage. Les très grandes images sont réduites à ~700px de large. Le pixel que vous cliquez vient du canvas affiché, pas de l'original. Pour une extraction pixel-perfect d'un élément minuscule, croppez avant l'upload.
  • HSL est dérivé, pas stocké. Le navigateur stocke RGB ; HSL est une autre façon de décrire la même couleur. Les round-trips peuvent introduire de légers drifts dus à l'arrondi.
